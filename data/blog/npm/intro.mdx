### semver 的  major.minor.patch  模型
    - `^`:表示最新的次版本，例如， ^1.0.4  可能会安装主版本系列  1  的最新次版本 1.3.0。
    - `~`:表示最新的补丁程序版本，与  ^  类似， 〜1.0.4  可能会安装次版本系列 1.0  的最新次版本1.0.7。

    所有这些确切的软件包版本都将记录在  package-lock.json  文件中

# package-lock.json
 - package-lock.json  通常是由  npm install  命令生成的
 - package.json  是通用的描述性标签，则  package-lock.json  是成分表
